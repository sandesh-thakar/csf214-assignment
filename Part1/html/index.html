Tasks

Task 1: Write a function to convert the infix propositional logic expression into a postfix propositional logic expression.
Task 2: Write a function to convert the postfix expression into a rooted binary parse tree.
Task 3: Write a function to traverse the parse tree to output the infix expression back by in-order traversal of the parse tree.

Assumptions

The expression should be perfectly paranthesized
Example 1: (a^b) is perfectly paranthesized while a^b is not
Example 2: ((~p)^q) is perfectly paranthesized while (~p^q) is not

Sample

Input
Enter infix expression: (((~p)^(qV(~p)))>p)
Output
Postfix expression is: p~qp~V^p>
Infix expression after inoder traversal of parse tree: (((~p)^(qV(~p)))>p)
